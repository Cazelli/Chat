# Chat

## Server Side

The server side code was made with the following technologies:

* ASP.NET Core
* SignalR
* Entity Framework
* Cqrs
* MediatR

## Client Side

It was developed two different clients one with Angular and another with React JS.

Also used the following technologies:

* SignalR client
* Bootstrap
* TypeScript
* Redux
