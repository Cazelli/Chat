# Chat

## Server Side

To developed the server side code I used the following technologies:

* ASP.NET Core
* SignalR
* Entity Framework
* Cqrs
* MediatR

## Client Side

The chat client's were developed in Angular and ReactJS. Also used the following technologies:

* TypeScript
* Redux
* Bootstrap
* SignalR client
